-OBJECTIVE-

This program is a minigame where you try to guess the password to log into a fake acocunt.
You are given the username, and that's it.

-ABOUT-

This program is not only made for fun but also to show off my abilities with working with web frameworks.
This program is created with Flask, and the languages used here are HTML, CSS, and Python.
This program allowed me to experiment with Flask and gain experience in web development, design, and back-end development.

-HOW TO RUN-

Run main.py on a computer that supports Python. Then click on the link that pops up on the command prompt.
